# Rawda – Smart Home Gardening Assistant

Rawda is a responsive front-end web application developed as part of the SWE363 course project. The platform is designed to assist home gardeners by providing an organized and user-friendly digital environment where they can manage plants, receive reminders, access plant-care guidance, communicate with gardening services, and interact with helpful smart features.


---

## Project Overview

Rawda aims to simplify plant care for beginner and home gardeners through a modern and accessible web interface. The application provides several interactive pages that support daily gardening activities, service requests, reminders, and informational guidance in a visually clean environment.

This repository contains the complete front-end implementation developed using React and Vite.

---

## Main Features

- Responsive and user-friendly interface
- Home page introducing the Rawda platform
- Gardener dashboard and plant management pages
- Interactive chatbot for plant-care assistance
- Watering reminder and scheduling interface
- Nearby gardening store browsing page
- Gardening community/forum pages
- Plant care service request forms
- Login and registration interfaces
- Profile and settings pages
- Error handling and fallback pages

---

## Technologies Used

- React.js
- Vite
- JavaScript
- Tailwind CSS
- Lucide React Icons (for ready icons instead of importing manually)
- React DOM

## Installation and Setup Guide

Follow the steps below to run the project locally.

### 1. Clone the Repository

```bash
https://github.com/Somaiakha/SWE-363-Rawda.git
```

### 2. navigate to the Project Folder
```bash
cd SWE363-Rawda
```

### 3. Install Project Dependencies
```bash
npm install
chmod +x node_modules/.bin/vite
npm run dev
```

### 4. Open the Local Development Link
```bash
http://localhost:5173
```
